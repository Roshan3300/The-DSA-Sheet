# Google - Software Engineer Interview Questions

## Common Question Patterns
- Graph algorithms and tree traversals
- Dynamic programming (complex variations)
- System design at scale
- Data structures optimization
- Algorithm complexity analysis

## Difficulty Level: Medium to Hard

## Interview Process
- Phone/Video Screen (1-2 rounds)
- On-site/Virtual Loop (4-5 rounds)
- Focus on problem-solving approach

## Tips
- Think out loud during problem solving
- Optimize for both time and space complexity
- Consider edge cases thoroughly
- Clean, readable code is essential

## Frequently Asked Topics
- Graph algorithms (shortest path, MST)
- Advanced tree problems
- String algorithms and pattern matching
- System design fundamentals
- Behavioral questions
