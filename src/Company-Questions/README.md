# Company-Specific Coding Questions

This section contains coding questions and solutions from various companies' recruitment processes.

## Companies Covered

### Service-Based Companies
- **TCS-Ninja/** - TCS Ninja recruitment questions
- **TCS-CodeVita/** - TCS CodeVita competition problems
- **Accenture/** - Accenture coding assessment questions
- **Tech-Mahindra/** - Tech Mahindra interview questions

### Product-Based Companies
- **Google/** - Google interview questions and coding challenges
- **Amazon/** - Amazon SDE interview questions
- **Microsoft/** - Microsoft coding interviews
- **NVIDIA/** - NVIDIA technical interviews
- **Oracle/** - Oracle database and coding questions
- **PayPal/** - PayPal software engineering questions

## Structure
Each company folder should contain:
- `easy/` - Easy level questions
- `medium/` - Medium level questions  
- `hard/` - Hard level questions
- `solutions/` - Solutions with explanations
- `notes.md` - Company-specific tips and patterns

## Usage
1. Navigate to the specific company folder
2. Practice questions by difficulty level
3. Review solutions and understand patterns
4. Take notes on common question types
